Installation of Bentō

The are some dependencies with other plug-ins, that should be installed before installing Bentō.
  • EMFText (from version 1.4.1) - http://www.emftext.org/
  • ATL (from version 3.0) - http://www.eclipse.org/atl/
  • Eclectic - See installation instructions

Download the packaged update site from one of the links at the left sidebar. Install with: 

Help -> Install New Software -> Add... -> Archive ...
It may be needed to uncheck "Group items by category" if nothing appears in the plug-in list.
